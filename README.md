# Duality Solver Mathematics

This project is designed to solve mathematical duality problems in a simple and structured way.

## Features
- Solves duality-based mathematical problems
- Accepts user input
- Gives computed output clearly

## How to Run
1. Make sure Python is installed
2. Open the project folder in VS Code
3. Run the main Python file

## Purpose
This project helps in understanding and solving duality problems more easily using programming.
